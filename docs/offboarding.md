# Vendor Offboarding

Use this checklist when a vendor engagement ends.

- Confirm the engagement has ended with the business owner.
- Disable any temporary supplier accounts.
- Remove remote support access packs from active distribution folders.
- Check whether the vendor created local files or exports that need review.
- Confirm no outstanding CRM, ticket, or invoice workflow remains.
- Record the closure in the service desk ticket.

If access was used during an incident or emergency support window, preserve logs before cleanup.
