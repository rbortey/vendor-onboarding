# Vendor Onboarding

Operational notes and templates for onboarding external suppliers, support partners, and short-term implementation vendors.

This repository is used by operations and IT support to keep lightweight onboarding material consistent across supplier engagements.

## Contents

- `docs/supplier-access.md` - access request and approval guidance
- `docs/remote-support-checklist.md` - remote support readiness checklist
- `forms/vendor-access-request.md` - request template for new vendor access
- `docs/offboarding.md` - closure and review checklist

## Handling

Do not place live production credentials in this repository. Temporary access details should be issued through the approved service desk workflow and reviewed after the engagement window closes.
